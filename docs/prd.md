# MEDIATOR — Product Requirements Document (PRD)

> 작성일: 2026-06-09  
> 스프린트: 1주 (v1.0)

---

## 1. 제품 목표

스타트업 팀의 회의를 AI가 실시간으로 중재한다.  
강제로 끼어들지 않는다. 존재를 알리고, 팀이 선택하게 한다.  
회의가 끝나면 결정 사항과 액션 아이템이 명확하게 나온다.

---

## 2. 타겟 사용자

- 소규모 스타트업 팀 (대면 회의 중심)
- 카카오톡 + 노션 사용 중
- 회의 중 주제 이탈, 감정 충돌, 결론 없이 끝나는 문제를 겪고 있음

---

## 3. v1.0 성공 기준 (1주 후)

| 기준 | 측정 방법 |
|---|---|
| 팀이 실제 회의에서 한 번 써봄 | 실사용 여부 |
| AI 신호 버튼이 자연스럽게 작동함 | 팀이 불편 없이 사용 |
| 음성 인식이 전보다 확실히 낫다는 체감 | 주관적 판단 |
| 회의 끝나고 액션 아이템이 명확하게 나옴 | 출력물 확인 |

## 4. v1.0 핵심 기능 (이번 1주)

### 4-1. AI 신호 버튼 (P0)

**현재:** AI 분석 결과가 채팅 패널에 자동으로 표시됨  
**변경:** AI가 할 말 있을 때 신호만 보내고, 팀이 자발적으로 확인

**동작 흐름:**
```
AI 중재 포인트 감지
    ↓
모든 기기 신호 버튼 활성화
 · 버튼 색 변화 (회색 → 보라)
 · 짧은 진동 (모바일)
 · 짧은 소리 (띵)
    ↓
팀원이 원할 때 탭
    ↓
채팅 패널에 AI 메시지 버블 표시
    ↓
버튼 원래 상태로 복귀
```

**규칙:**
- 누르지 않으면 메시지 안 보임 (완전 opt-in)
- 모든 기기에 동시에 신호, 각자 독립적으로 확인
- 읽지 않은 신호 수 뱃지 표시 (예: 🔔 2)
- 회의 종료 시 안 읽은 메시지 모두 자동 공개

**트리거 조건 (기존 유지):**
- transcript 3개마다 자동 분석
- 주제 이탈 / 감정 충돌 / 결론 지연 감지 시 긴급 체크

---

### 4-2. 음성 인식 품질 개선 (P0)

**두 단계로 구현:**

#### Step 1 — Claude 교정 (즉시)
발화가 확정되면 Claude가 맞춤법/문맥을 다듬어서 저장

```
STT 확정 텍스트
    ↓
Claude Haiku: "회의 발화 텍스트를 자연스럽게 교정해줘 (원문 의미 유지)"
    ↓
교정된 텍스트로 transcript 저장
```

- 추가 비용: 거의 없음 (발화 하나 ≈ $0.0001)
- 개발: 1일

#### Step 2 — Deepgram 실시간 STT (3일 내)
Web Speech API → Deepgram Nova-2로 교체

```
마이크 오디오 스트림
    ↓
Deepgram WebSocket (실시간)
    ↓
한국어 인식 (Web Speech보다 정확)
    ↓
기존 STT 흐름과 동일하게 처리
```

- 비용: $0.0043/분 → 1시간 회의 ≈ $0.26
- 무료 크레딧 $200 제공 (수백 시간 무료)
- 개발: 2-3일

**다중 발화자 동시 발화 문제:**
- 기술적 한계: 하나의 마이크로 두 목소리 분리 불가
- 해결책: 각자 자기 기기 마이크 사용 유도 (이미 멀티기기 구조)
- UI: "각자 폰이나 노트북에서 말하면 더 잘 인식돼요" 안내 문구 추가

---

### 4-3. 회의 후 액션 아이템 명확화 (P0)

**현재:** 회의 요약이 긴 텍스트 덩어리로 출력  
**변경:** 액션 아이템을 담당자 + 기한 형태로 구조화

**출력 형태:**
```
📋 결정 사항
· 다음 분기 마케팅 예산 500만원으로 확정
· 런칭일 6월 20일로 연기

✅ 액션 아이템
· 김건주 — 디자인 시안 완성 (6월 13일까지)
· 이수진 — 개발자 미팅 일정 잡기 (이번 주)
· 박민준 — 투자자 업데이트 메일 발송 (오늘)

⚠️ 미결 사항
· 가격 정책 최종 결정 (다음 회의에서)
```

**Claude 프롬프트 강화:**
- 발화 내용에서 "~하기로", "~가 맡기로", "~까지" 패턴 추출
- 이름 + 동사 + 기한이 있는 발화를 액션 아이템으로 분류
- 불명확하면 "담당자 미정" 으로 표시 (생략 X)

## 6. Non-goals (v1.0에서 하지 않는 것)

- 노션 연동
- 카카오톡 공유
- 서버 사이드 화자 분리 (diarization)
- 모바일 앱 (PWA 유지)
- 회원가입 / 로그인

## 7. 기술 제약 및 리스크

| 리스크 | 대응 |
|---|---|
| Deepgram 한국어 품질이 기대 이하 | Claude 교정(Step 1)으로 선 보완, Clova 검토 |
| AI 신호 버튼 무시 → AI 개입 효과 없음 | 읽지 않은 신호 뱃지 + 회의 종료 시 자동 공개 |
| 액션 아이템 자동 추출 부정확 | 수동 편집 기능 추가 (v1.1) |
| Firebase 비용 | 세션 24시간 후 자동 삭제 (v1.1) |
