## 1. 문제 정의

### 핵심 페인포인트
스타트업 팀 내 소통 문제. 세 가지 레이어에서 동시에 발생한다.

| 타이밍 | 문제 |
|---|---|
| 회의 중 | 주제 이탈, 감정 충돌, 결론 없이 끝남 |
| 회의 후 | 결정 사항이 어영부영 넘어감, 액션 아이템 불명확 |
| 회의 전 | 팀원들이 서로 다른 전제로 회의에 들어옴 |

### 우선순위
**1순위**: 회의 중 실시간 AI 중재  
**2순위**: 회의 전 전제 정렬 (pre-meeting alignment)  
**3순위**: 회의 후 결과 정리 및 공유

### 타겟 사용자
- 스타트업 소규모 팀 (대면 회의 중심)
- 현재 카카오톡 + 노션 사용 중
- 별도 회의 도구 없이 구두로 진행하다가 결론이 흐지부지되는 경험 보유

---

## 2. 솔루션 개요

**MEDIATOR**: AI 실시간 회의 중재자

회의를 녹음/분석하고, 팀이 자발적으로 확인할 수 있는 방식으로 AI 개입을 제공한다.  
강제로 끊지 않는다. 존재를 알리고, 팀이 선택하게 한다.

## 3. 핵심 기능 명세

### 3-1. 회의 전: Pre-meeting Setup

- **호스트 혼자 입력** (팀원은 물리적으로 같은 공간에 있으므로 실시간 반영 가능)
- AI가 역질문하며 4가지 맥락 파악:
  1. 회의 목적
  2. 현재 상황/배경
  3. 원하는 결과물
  4. 예상 갈등 지점
- 4가지 모두 파악되면 ✅ 신호 → 회의 시작
- 다른 기기(팀원)에서는 AI 셋업 대화가 실시간으로 보임 (Firebase 구독)

### 3-2. 회의 중: 실시간 AI 중재

#### AI 개입 트리거
| 상황 | 설명 |
|---|---|
| C. 주제 이탈 | 대화가 원래 의제와 다른 방향으로 흘러갈 때 |
| D. 의견 충돌 | 감정적 반응, 반복되는 논쟁 감지 |
| E. 결론 지연 | 결정해야 할 시점인데 아무도 정리 못 하고 있을 때 |

#### AI 신호 버튼 UX
```
AI가 할 말 있음 감지
        ↓
모든 기기에 신호 버튼 등장 (색 변화 + 짧은 소리)
        ↓
각자 타이밍에 자발적으로 탭
        ↓
채팅 패널에 AI 메시지 버블로 표시
```

- **비침투적**: 말 중간에 끊지 않음
- **자발적**: 누르고 싶지 않으면 무시 가능
- **동기화**: 모든 팀원 기기에 동시 신호, 각자 독립적으로 확인

#### AI 중재 스타일
**충돌 상황 (D)**: 중립 정리 방식
> "두 의견을 정리하면, A는 [의견 A]이고 B는 [의견 B]입니다. 어떤 기준으로 결정할까요?"

감정 편들기 없이 양쪽 입장을 명확히 정리하고, 결정 기준을 팀에 돌려줌.

**주제 이탈 (C)**: 원래 의제로 복귀 유도
> "현재 대화가 [원래 주제]에서 벗어난 것 같아요. 이 이야기는 별도로 다루고 원래 주제로 돌아갈까요?"

**결론 지연 (E)**: 결정 촉구
> "지금까지 논의된 내용을 바탕으로 결론을 내릴 수 있을 것 같아요. [요약]. 이 방향으로 결정할까요?"

### 3-3. 회의 후: 결과 정리 및 공유

#### 출력물 (3가지 필수)
1. **결정 사항** — 회의에서 확정된 것들
2. **액션 아이템** — 담당자 + 기한 포함
3. **미해결 사항** — 다음 회의로 넘겨야 할 것

#### 공유 방식
| 방식 | 우선순위 |
|---|---|
| 카카오톡 자동 전송 (단체 채팅방) | 1순위 |
| 복사 버튼 (클립보드) | 1순위 |
| 링크 공유 (결과 페이지 URL) | 1순위 |
| 노션 자동 페이지 생성 | 2순위 |

---

## 4. 성공 지표 (팀 기준)

회의가 "잘 됐다"는 느낌의 조건:
- ✅ 결정된 사항이 명확하게 정리됨
- ✅ 각자 할 일(액션 아이템)이 명확히 나옴
- ✅ 의견 충돌이 있었더라도 결론이 남
